# Campaign Tracker

A simple web application to manage marketing campaigns in real-time.  
Users can **add, view, update, and delete campaigns** easily.

---

## Features

- Add a new campaign with name, client, start date, and status.
- View all campaigns in a clean table format.
- Update campaign status (Active / Paused / Completed).
- Delete a campaign if needed.

---

## Tech Stack

- **Frontend:** React  
- **Backend:** Node.js + Express  
- **Database:** MongoDB
